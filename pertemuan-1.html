BOM (Browser Object Model) kita dapat memberikan perintah-perintah khusus ke browser, misalnya melalui sebuah 'atribut' khusu milik browser yaknik 'window' sehingga kita bisa membuat browser menampilkan pesan pop-up. Caranya yakni menjalankan method alert() pada console milik browser. Seperti berikut contohnya:

Selain method alert(), objek window juga memiliki method-method lainnya, seperti prompt, console, dsb. Tenang, kita akan berjelajah lebih jauh pada modul-modul selanjutnya.

Cara kedua adalah DOM. DOM sama seperti BOM. Perbedaanya adalah kita menggunakan global objek bernama document. Melalui global objek ini, kita bisa menangkap seluruh elemen dalam dokumen HTML guna memanipulasi konten HTML melalui method getElementById. Method ini akan menangkap elemen berdasarkan value dari atribut id. Sebagai cotnoh, ktia mengubah konten elemen HTML berikut ini.

Pada contoh yang kita pelajari di atas merupakan contoh yang sangat sederhana. Kita dapat melakukan lebih banyak hal lainnya, seperti mengubah konten elemen, memberikan event tertentu pada elemen, dan sebagainya.

Melalui contoh di atas, kita telah menggunakan console milik browser untuk menjalankan kode program JavaScript. Tentinya jika kita mengembangkan website tidak menggunakan pendekatan seperti ini. Ada cari lain dalam menulis kode JavaScript, yakni melalui berkas HTML secara langsung.
