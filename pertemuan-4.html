Pendahuluan Browser Object Model (BOM)

Setiap browser menyediakan apa yang disebut Browser Object Model atau BOM yang bisa kita gunakan dalam kode JavaScript kita. Dalam browser, BOM ini diwakilkan oleh sebuah objek bernama window. Mari kita ketik window ada console browser dan lihat method apa saja yang tersedia melalui objek ini:

Melalui objek window inilah kode JavaScript kita bisa mengakses berbagai method dan atribut. Akan tetapi, Anda tidak diharapkan untuk menghafal semuanya karena materi berikutnya kita cukup membahas 3 method paling dasar yakni alert(), prompt() dan objek console.

Macam-macam anggota dari BOM

Pada materi ini, kita akan membahas lebih dalam mengenai method alert

Method Alert

Method alert() merupakan salah satu method BOM yang menerima satu parameter opsional berupa data string yang akan ditampilkan melalui sebuah pop-up browser. Contohnya, jika kita ingin menampilkan sebuah pesan berisi stirng "Halo, user!" menggunakan console browser, caranya sebagai berikut:

Tampilannya akan terlihat seperti berikut.

Sama seperti memanggil method pada umumnya pada JavaScript, bukan? Kita juga bisa memasukkan variabel sebagai parameternya seperti berikut.

Jika dijalankan pada console browser, hasilnya akan seperti berikut.

Kita bisa menampilkan pesan yang telah ditentukan melalui JavaScript. Namun, ada satu hal yang perlu kita ingat saat menggunakan alert(), yakni ketika pop-up muncul, user tidak bisa berinteraksi dengan bagian lain dari halaman web sampai pop-up tersebut ditutup.

Method Prompt

Method ini akan menampilkan sebuah dialog browser yang meminta user untuk mengisi kolom inputnya. Karena bisa menangkap input dari user, method ini memiliki lebih banyak fungsionalitas ketimbang alert()

Nilai kembalian dari prompt

Jika kita lihat, sebenarnya method alert() mengembalikan sebuah value berupa undefined. Namun, prompt() akan mengembalikan sebuah nilai sesuai dengan inputan user. Sebagai contoh, mari kita jalankan kode berikut pada console browser.

Lalu, coba kita tampilkan isi dari variabel pesanInput, maka hasilnya pasti akan sesuai dengan yang kita input pada pop-up.

Kita bisa melihat bahwa nilai yan tersimpan di variabel pesanInput sesuai dengan input yang diberikan user, yakni "Aku suka belajar koding". Jika user tidak mengisi apa pun dan tetap menemkan tombol "OK", maka nilai yang diberikan hanya string kosong.

Pada contoh di atas, kita telah menekan tombol "OK", tetapi apa yang terjadi jika kita menekan tombol "Cancel"? Jika kita menekan tombol "Cancel", justru nilai yang akna dikembalikan adalah null. Silahkan coba kembali contoh di atas, nanti nilai variabel pesanInput akan bernilai null

Ada satu hal yang perlu kita ingat selama menggunakan prompt(), yakni nilai apa pu yang dimasukkan oleh user akan diproses dan dikembalikan menjadi data string. Silahkan jalankan kode berikut pada console browser.

Ketika kita menampilkan tipe data dari variabel pesanInput, maka sudah pasti tipe datanya adalah string, walau yang kita masukkan berupa angka.

Hal ini perlu kita ingat baik-baik. Kode tidak bisa menduga bahwa variabel pesanInput merupakan string atau number, ia akan tetap mengeluarkan string. Meskipun demikian , kita bisa cast atau parse hasilnya ke tipe data tertenti. Namun, harap lebih teliti karena jika kita salah melakukan parse ke tipe data maka akan menghasilkan NaN. Sebagai contoh, kita ingin mendapatkan value number dari prompt() maka kita bisa menggunakan cara ini.

Jika kita periksa tipe data dari hasilnya, maka hasilnya berupa number. Selain Number(), kita juga bisa menggunakan function parseInt() untuk melakukan parsing data string ke number.

Memberikan Nilai Default untuk prompt

Terdapat satu parameter lagi yang bisa kita masukkan ke prompt(), dimana parameter ini berguna sebagai nilai alternatif. Mari jika jalankan kode berikut.

Jika kita jalankan pada browser, maka kolom input yang muncul pada dialog box akan memiliki data secara otomatis

Terdapat satu hal penting, yaitu dialog box dari prompt() muncul, maka kita tidak bisa berinteraksi dengan komponen lain pada halaman web sampai dialog box tersebut ditutup. Hal ini sama halnya dengan alert()
